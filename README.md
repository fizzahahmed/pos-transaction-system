# POS Transaction System

## 📌 Overview

The POS Transaction System is a desktop application developed using C# Windows Forms. It is designed to simplify retail sales by allowing users to manage products, process customer transactions, and generate bills through an easy-to-use graphical interface.

---

## ✨ Features

- Add and manage products
- Process customer transactions
- Calculate total bill
- User-friendly graphical interface
- Efficient transaction handling

---

## 🛠️ Technologies Used

- C#
- Windows Forms
- .NET
- Visual Studio

---

## 🚀 How to Run

1. Open the solution in Visual Studio.
2. Restore any required packages.
3. Build the project.
4. Run the application.
